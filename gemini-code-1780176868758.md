# Product Requirements Document (PRD): AdmiAPP

**Objetivo del Proyecto:** Desarrollar una aplicación de administración completa orientada a la gestión operativa, financiera, estratégica y de seguridad. El sistema requiere alta precisión en cálculos de costos, trazabilidad de inventarios y control estricto de permisos.

---

## 1. Requerimientos Operativos
Define la gestión del día a día y el flujo central de datos del negocio.

| Módulo | Atributos y Requisitos |
| :--- | :--- |
| **Productos** | Nombre, Categoría, Precio de venta, Canal de venta (local/mayorista), Porcentaje de margen, Estado activo/inactivo. |
| **Recetas / Fórmulas** | Insumos requeridos, Cantidades exactas, Porcentaje de merma, Rendimiento, Cálculo de costo automático por unidad. |
| **Insumos** | Precio de compra, Proveedor, Lote, Fecha de compra, Stock actual, Historial de variación de precios. |
| **Producción** | Fecha, Producto fabricado, Cantidad, Lote asignado, Costo total del lote, Costo unitario real. |
| **Stock General** | Inventario de productos terminados, Inventario de insumos, Alertas de stock mínimo, Control de vencimientos. |
| **Ventas** | Fecha, Producto vendido, Cantidad, Medio de pago, Canal utilizado, Descuento aplicado, Margen bruto por operación. |
| **Gestión de Clientes** | Historial de compra, Control de deuda, Frecuencia de compra, Ranking de clientes. |
| **Gestión de Proveedores** | Historial de compras, Deudas pendientes, Variación de precios. |

---

## 2. Requerimientos Financieros
Control de flujos monetarios y estructura integral de costos.

| Módulo | Componentes a Trackear |
| :--- | :--- |
| **Egresos Regulares** | Gastos fijos (Alquiler, Luz, Gas, Internet, Sueldos, Servicios). Gastos variables (Packaging, Fletes, Comisiones, Publicidad). |
| **Carga Impositiva** | Monotributo, Ingresos Brutos, Retenciones. Debe incluir cálculo automático sobre facturación. |
| **Costos Laborales y Activos** | Mano de obra real: Costo por hora, Horas por producción, Costo laboral por producto. Amortización: Equipos, Vehículos, Vida útil, Costo mensual imputado. |
| **Cuentas Corrientes** | Cuentas por cobrar: Clientes con deuda, Vencimientos. Cuentas por pagar: Proveedores, Impuestos futuros, Compromisos. |
| **Flujo de Caja** | Registro de entradas, Registro de salidas, Saldo real, Proyecciones a 30 y 60 días. |
